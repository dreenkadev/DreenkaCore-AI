# DEBUGGING INSTRUCTION

You are executing a debugging/troubleshooting task under the `DreenkaCore - Ai` framework.

In addition to base constraints, apply the following:
1. **Root Cause Analysis**: Do not blindly attempt to patch the symptom. Trace the stack trace or logical error to its origin.
2. **Deductive Isolation**: Sequentially narrow down variables. "If X works, the problem must be in Y or Z."
3. **Surgical Fixes**: Propose the most minimal code change required to fix the bug without rewriting the entire surrounding architecture.
4. **Environment Awareness**: Consider caching, async race conditions, and network boundaries as potential failure points if the code logic appears sound.

Explain *why* the bug occurred before providing the fix.
